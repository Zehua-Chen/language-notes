# System.Numerics

The namespace provides

- SIMD enabled types
  - `Vector4`
  - `Vector3`
  - `Vector2`
  - `Vector<T>`
  - `Quaternion`
  - `Plane`
  - `Matrix3x2`
  - `Matrix4x4`
- Big integers
- Complex numbers

# SIMD

- Vector operations are provided as static methods in, the related vector types
- Vector types are pretty printed
